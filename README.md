# jade

Este proyecto es una aplicación web dedicada a celebrar aniversarios, con cuenta atrás, mensajes generados por IA (Gemini) y galería de recuerdos.

## Comandos para subir al repositorio:

```bash
echo "# jade" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/joseleonelmorenopernia-spec/jade.git
git push -u origin main
```

## Características:
- Cuenta atrás para el día 27.
- Mensajes especiales de aniversario (IA).
- Lógica especial para el 27 de febrero (1er Año).
- Galería de fotos personalizada.
- Notificaciones PWA.
